# Food-Recipes-App
This is a web app to search recipes

- Users can search a recipe by food name

- Displays some popular latest meals


## Deployed Link

- [Food Recipes App](https://chic-chimera-75d0d0.netlify.app)

## Tech Stack

- HTML | CSS | JavaScript


## API Reference

### API used
